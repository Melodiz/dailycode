# [Тандемная конкатенация](link)

## Description

Тандем $-$ это строка четной длины, у которой левая и правая половины равны.

Дан набор из $n$ строк ($1 \le n \le 10^5$).
Найдите среди них такие пары, которые при конкатенации дают тандем.
Более формально, найдите все пары ($i$, $j$) такие, что $i \ne j$ и строка $s_i$+$s_j$ является тандемом.

Выведите все упорядоченные пары индексов (нумерация с единицы).
### Input Format:

В первой строке дано целое число $n$ ($1 \le n \le 100\,000$) $-$ количество строк.

Далее в $n$ строках записано по одному слову. Длина каждого слова от $1$ до $10$. 
Слова состоят из маленьких букв английского алфавита.

### Output Format:

Выведите все упорядоченные пары индексов (нумерация с единицы).
