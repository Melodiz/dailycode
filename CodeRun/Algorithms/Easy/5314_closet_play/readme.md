# [5314. Игра из чулана](https://coderun.yandex.ru/problem/closet_play)

## Description

Кодерун помогал другу с уборкой после праздников и нашёл в чулане игру. От правил для игры осталась лишь одна фраза:

«Чтобы начать играть, игроки должны иметь две одинаковые строки длины ровно $k$».

Но пока у каждого игрока есть только какая-то своя строка. Им интересно, смогут ли они свои строки сделать подходящими для начала игры.

Кодерун решил, что просто вырежет подстроку длины $k$ из своей строки $s$. Эту подстроку он и будет использовать, чтобы начать игру. Его друг воспринял игру как вызов, а потому решил, что он может вырезать из своей строки вообще любые $k$ символов, а затем склеить их в одну строку в каком-то порядке. Так он получит строку длины $k$, которую и будет использовать, чтобы начать игру.

Помогите друзьям узнать, смогут ли они начать игру.

### Input Format:

В первой строке входных данных находится целое число $k$ — требуемая длина строк, необходимых для игры ($1 \le k \le 3 \cdot 10^5$). В следующих двух строках находятся непустые строки $s$ и $t$ — строки, которые изначально есть у Кодеруна и его друга, соответственно. Строки состоят только из маленьких латинских букв, а их длина не превосходит $3 \cdot 10^5$.

### Output Format:

Если ребята смогут начать игру, то выведите единственную строку «‎YES»‎, без кавычек. Иначе выведите «‎NO»‎, без кавычек.
